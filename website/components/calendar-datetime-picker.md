## API

| 属性    | 说明     | 类型                          | 默认值            |
|-------|--------|-----------------------------|----------------|
| value | 当前值    | `DateTime? `                | DateTime.now() |
| onOk  | 确定按钮回调 | `ValueChanged<DateTime?>? ` |                |
