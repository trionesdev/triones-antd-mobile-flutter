## API

| 属性      | 说明     | 类型     | 默认值  |
|---------|--------|--------|------|
| src     | 图片地址   | String | -    |
| width   | 宽度     | double | -    |
| height  | 高度     | double | -    |
| preview | 是否可以预览 | bool   | true |
