## API

| 属性              | 说明        | 类型          | 默认值 |
|-----------------|-----------|-------------|-----|
| backgroundColor | 背景色       | `Color`     |     |
| label           | 标签，用于显示文字 | `Widget`    |     |
| count           | 数字        | `int`       |     |
| maxCount        | 最大显示数字    | `int`       |     |
| alignment       | 对其方式      | `Alignment` |     |
| offset          | 偏移        | `Offset`    |     |
| textStyle       | 文本样式      | `TextStyle` |     |
